What Exactly is NoteHub?
NoteHub is a note-sharing web app meant for users of all ages.

NoteHub is a project which aims to launch a secure, safe, reliable platform to share, edit and view notes for everyone, with high-quality notes for all to see, vetted by intelligent algorithms. It also posseses a curated gallery for signed-in users.

Who Created This?
NoteHub is a project made by four web programmers.
