What Exactly is NoteHub?
NoteHub is a note-sharing web app meant for users of all ages.

The NoteHub team aims to launch a secure, safe, reliable platform to share, edit and view notes for everyone, with high-quality notes for all to see, vetted by only the best intelligent algorithms. It also posseses a curated gallery exclusively for signed-in users.

Who Created This?
NoteHub is a project made by four web programmers.

It's currently an Infocomm school project in Hwa Chong Institution, being updated and improved on every single day. A huge thanks to 000webhost for hosting our dynamic web content and GitHub Pages for hosting our static web site.
